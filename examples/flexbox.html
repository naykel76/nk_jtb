💡 Grid alignment cheat:
• justify-* = horizontal (row axis)
• align-* = vertical (column axis)
• *-items = align content INSIDE cells
• *-content = align the ENTIRE grid
• place-* = shorthand (vertical / horizontal)

🎯 Quick Memory Aids

FLEXBOX

justify-content
Main axis (usually →)

align-items
Cross axis (usually ↓)

align-self
Individual item override

GRID

justify-* = Horizontal →

align-* = Vertical ↓

*-items = Inside cells

*-content = Entire grid

🚀 Center anything instantly:
Flexbox: display: flex; justify-content: center; align-items: center;
Grid: display: grid; place-items: center;
